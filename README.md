# ARI Korean OS

Goal-based Korean lesson creation engine for speaking-focused teaching.

## Features

- 📊 Teacher dashboard: 수업 생성 + 학생별 Natural Upgrade 기록
- 🎯 Lesson assembler: Student → Scenario → Stage → Auto-linked Grammar/Patterns/Vocab
- 👨‍🎓 Student view: 오늘의 미션 + 표현 은행 + 숙제 체크
- 📱 Responsive: PC + Mobile (bottom tab navigation)

## How to Use

1. Open `ari-korean-os.html` in browser
2. Add students & seed data via Resource Library
3. Create lessons by clicking Student → Scenario → Stage
4. Record student sentences in Natural Upgrade section
5. Share student view link with learners
